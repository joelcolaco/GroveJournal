# Grove Journal Website

Static public website for Grove Journal.

Planned production URL: `https://www.joelcolaco.com/GroveJournal`

## Goals

- Simple GitHub Pages site
- Light Grove Journal visual style
- English, Spanish, French, and German pages
- Privacy Policy and Terms of Service pages
- GitHub Issue Forms for feature requests and fallback bug reports

## Notes

This repository intentionally starts as plain static HTML/CSS so it can publish directly from GitHub Pages without a build step.
